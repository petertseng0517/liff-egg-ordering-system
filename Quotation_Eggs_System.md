# 🐔 土雞蛋訂購系統 - 專案報價與營運成本評估

本文件詳細列出「土雞蛋訂購系統」的開發內容、功能範疇，以及系統上線後的預估營運成本，供業主參考與決策。

---

## 1. 系統功能範疇 (Scope of Work)

本專案包含以下三大核心模組的開發與建置：

### A. 客戶端訂購系統 (LINE LIFF)
*   **會員系統**：自動綁定 LINE 帳號，支援姓名、電話、出生日期及 **雙配送地址** 註冊。
*   **商品訂購**：直觀的選購介面，支援 **優惠組合自動換算** (如：選購「11盤優惠組」系統自動記錄為 11 盤)。
*   **訂單管理**：即時查詢歷史訂單、查看訂單狀態 (處理中/配送中/已完成) 及 **付款狀態** (未付款/已付款)。
*   **自動通知**：下單成功後，自動收到 LINE 確認訊息。

### B. 管理員專屬後台 (Admin Dashboard)
*   **安全登入**：具備密碼保護的後台入口。
*   **訂單總覽**：支援日期篩選 (今日/本週/本月)、關鍵字搜尋、狀態篩選。
*   **分批出貨管理**：針對農產品特性設計，支援單筆訂單 **分多次出貨**，系統自動計算剩餘盤數並更新狀態。
*   **金流管理**：獨立的付款狀態標記，方便對帳。
*   **推播通知**：管理員更新出貨進度時，系統自動發送 LINE 訊息通知客戶。

### C. 系統架構與資料庫
*   **雲端部署**：系統託管於 Render 雲端平台，確保穩定運行。
*   **資料庫**：使用 Google Sheets 作為後端資料庫，方便業主直接查看或匯出資料，且維護成本極低。

此費用包含上述所有功能的程式開發、系統架設、LINE 官方帳號串接及上線測試。

| 項目 | 說明 | 費用 (新台幣) |
| :--- | :--- | :--- |
| **系統開發費** | 前端介面、後端邏輯、資料庫串接 | **NT$ 38,000** |
| **系統建置費** | 伺服器環境設定、LINE API 申請與設定 | **NT$ 5,000** |
| **教育訓練** | 後台操作教學、文件交付 | **贈送** |
| **總計** | | **NT$ 43,000** |

*(註：以上價格為預估參考，可依實際合作細節調整)*

## 3. 每月營運成本 (Monthly Recurring Cost)

系統上線後，需支付給雲端服務商的租用費用。建議由業主信用卡直接綁定支付，以確保帳號所有權。

| 項目 | 方案建議 | 預估費用 | 說明 |
| :--- | :--- | :--- | :--- |
| **雲端主機 (Render)** | **Starter Plan** | **約 NT$ 230 / 月** ($7 USD) | **強烈建議**。確保網站不休眠，客戶隨點隨開，體驗流暢。若選免費版，閒置後需等待 30 秒喚醒。 |
| **資料庫 API** | Google Sheets | **$0** | 在合理用量下完全免費。 |
| **LINE 官方帳號** | **輕用量 (免費)** | **$0** | 每月 500 則免費群發訊息。約可支援每月 150 張訂單。 |
| **LINE 官方帳號** | 中用量 (升級) | NT$ 800 / 月 | **(選購)** 若生意成長，每月訂單超過 150 張時才需考慮升級。 |
| **網域名稱** | 專屬網址 | 約 NT$ 50 / 月 | **(選購)** 如 `happy-eggs.com`，費用約 NT$ 600/年。 |

### 💡 營運成本總結
*   **初期預估**：**約 NT$ 230 / 月** (僅需支付主機費)。
*   **未來擴充**：若訂單量大增，最高約 NT$ 1,000 / 月 (主機 + LINE 月費)。

---

## 4. 結語

本方案採用 **「輕量化架構」** (Render + Google Sheets)，在滿足完整商業邏輯 (訂購、分批出貨、對帳) 的同時，將長期維護成本降至最低。對於農產品銷售而言，是兼具專業形象與成本效益的最佳選擇。
